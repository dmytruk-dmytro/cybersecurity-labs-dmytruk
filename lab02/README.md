# Лабораторна робота №2: Захист особистих повідомлень

## Опис

У цій папці розміщено вихідний код програмного засобу для криптографічного захисту текстових повідомлень. Програма демонструє роботу класичних алгоритмів шифрування та порівнює їх стійкість.

* `main.py` - Основний код програми (клас `CryptoLab`).
* `requirements.txt` - Список бібліотек для встановлення.

## Технічне завдання

Скрипт `main.py` реалізує систему захищеного листування з такими функціями:

* **Реалізація алгоритмів:** Шифр Цезаря (моноалфавітна підстановка) та Шифр Віженера (поліалфавітна підстановка).
* **Адаптація під мову:** Повна підтримка українського алфавіту (33 літери).
* **Персоналізація ключів:** Автоматична генерація ключів шифрування на основі ПІБ та дати народження користувача.
* **Криптоаналіз:** Модуль для демонстрації вразливості шифру Цезаря методом повного перебору (Brute Force).
* **Візуалізація:** Побудова графіків частотного аналізу для порівняння стійкості шифрів.

## Технології

* **Мова:** Python 3.10+
* **Інтерфейс:** CLI (Командний рядок)
* **Бібліотеки:** `matplotlib` (для візуалізації графіків)

## Запуск програми

Для коректної роботи візуалізації (графіків) програма потребує встановлення бібліотеки `matplotlib`.

1.  Переконайтесь, що у вас встановлено **Python 3.x**.
2.  Перейдіть до папки проєкту (`lab02`) у вашому терміналі.
3.  Встановіть необхідні залежності:
    ```bash
    pip install -r requirements.txt
    ```
    *(або просто `pip install matplotlib`)*

4.  Запустіть скрипт:
    ```bash
    python main.py
    ```

5.  Далі дотримуйтесь інструкцій у консолі:
    * Введіть прізвище (для ключа Віженера).
    * Введіть дату народження (для зсуву Цезаря).
    * Введіть текст українською мовою для шифрування.
