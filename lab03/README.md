# Лабораторна робота №3: Невидимі дані у власних файлах

## Опис

У цій папці розміщено вихідний код CLI-утиліти для стеганографічного приховування та виявлення текстової інформації в зображеннях. Програма реалізує метод **LSB (Least Significant Bit)** з додатковим криптографічним захистом.

Архітектура проєкту — монолітна (`main.py`), що забезпечує портативність та зручність використання.

## Технічне завдання та Функціонал

Скрипт `main.py` реалізує повний цикл стеганографічної обробки:

* **Нативна LSB-логіка:** Власна реалізація алгоритму без використання готових стеганографічних бібліотек. Маніпуляція бітами відбувається напряму.
* **Багатоканальний запис:** Використовуються всі 3 колірні канали (RGB) пікселів, що збільшує ємність контейнера.
* **Гібридний захист:**
    * **Шифрування:** Текст попередньо шифрується алгоритмом **Віженера** (ключ задає користувач).
    * **Маркування:** Додається спеціальна сигнатура `||[EOF]||` для коректного визначення кінця повідомлення та відсіювання "цифрового шуму".
* **Data Preservation:** Система "розумного" збереження файлів (автоматичне створення імен `_encoded (N).png`), щоб не перезаписати оригінал.

## Технології

* **Мова:** Python 3.10+
* **Інтерфейс:** CLI (Командний рядок)
* **Залежності:** `Pillow` (тільки для доступу до пікселів зображення)

## Запуск програми

1.  Переконайтесь, що у вас встановлено **Python 3.x**.
2.  Перейдіть до папки проєкту (`lab03`) у терміналі.
3.  Встановіть залежність `Pillow`:
    ```bash
    pip install -r requirements.txt
    ```
4.  Підготуйте будь-яке зображення (наприклад, `image.png`) у папці з проєктом.
5.  Запустіть скрипт:
    ```bash
    python main.py
    ```

## Як користуватися

У меню програми доступні дві основні опції:

1.  **Приховати (Encode):**
    * Введіть шлях до оригінального зображення.
    * Введіть секретне повідомлення.
    * Введіть ключ для шифру Віженера.
    * *Результат:* Створиться новий файл (напр., `image_encoded.png`).

2.  **Витягти (Decode):**
    * Введіть шлях до файлу з секретом.
    * Введіть ключ для дешифрування.
    * *Результат:* Програма виведе приховане повідомлення або повідомить про помилку.
