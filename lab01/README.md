# Лабораторна робота №1: Аудит власної цифрової безпеки

## Опис

У цій папці розміщено вихідний код, який аналізує, наскільки надійним є пароль, враховуючи персональні дані.

* `main.py` - Вихідний код програми.
* `Скріншоти` - Декілька скріншотів, що показують, як працює програма.

## Технічне завдання

Скрипт `main.py` реалізує аналізатор безпеки паролів. Він оцінює паролі за критеріями, які були у методичних рекомендаціях:

* Оцінка складності (довжина та різноманітність символів).
* Аналіз зв'язку пароля з особистими даними (ПІБ, дата народження).
* Надання порад для покращення безпеки.

**Технологія:** Python 3 (CLI).

## Запуск програми

Програма не потребує встановлення жодних сторонніх бібліотек чи фреймворків.

1.  Переконайтесь, що у вас встановлено **Python 3.x**.
2.  Перейдіть до цієї папки (`lab01`) у вашому терміналі.
3.  Запустіть скрипт:

    ```bash
    python main.py
    ```

4.  Далі просто дотримуйтесь інструкцій у консолі: потрібно буде ввести Ім'я, Прізвище, Дату народження (ДД.ММ.РРРР) та пароль для перевірки.
