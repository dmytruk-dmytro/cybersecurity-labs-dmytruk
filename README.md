# cybersecurity-labs-dmytruk

## Лабораторні роботи з курсу «Безпека програм та даних»

Даний репозиторій є збіркою виконаних лабораторних робіт з навчальної дисципліни «Безпека програм та даних». Тут представлені як вихідні коди програм, так і супровідні матеріали, включаючи інструкції та скріншоти.

* **Виконавець:** Дмитрук Д. А.
* **Навчальна група:** 6.04.121.013.22.2
* **Університет:** ХНЕУ ім. С. Кузнеця

## Структура репозиторію

Вміст репозиторію організовано за папками. Кожна папка відповідає окремій лабораторній роботі згідно з навчальним планом.

| Папка | Назва лабораторної роботи | Вміст папки |
| :--- | :--- | :--- |
| `lab01` | Аудит власної цифрової безпеки | `main.py` (вихідний код), `README.md` (інструкції), Скріншоти |
| `lab02` | Захист особистих повідомлень | `main.py` (вихідний код), `README.md` (інструкції), `requirements.txt` (бібліотеки), Скріншоти |
| `lab03` | Невидимі дані у власних файлах | *(В процесі)* |
| `lab04` | Особистий цифровий підпис | *(В процесі)* |
| `lab05` | Захищена електронна пошта | *(В процесі)* |
| `lab06` | Етичний хакінг власного застосунку | *(В процесі)* |
| `lab07` | Комплексний захист особистого проєкту | *(В процесі)* |

## Інструкції з використання

1.  **Клонування репозиторію**

    Для отримання локальної копії проєкту, виконайте команду:
    ```bash
    git clone https://github.com/dmytruk-dmytro/cybersecurity-labs-dmytruk.git
    ```

2.  **Запуск програм**

    Кожна лабораторна робота є самостійним проєктом. Для запуску будь-якої програми необхідно перейти до її директорії (наприклад, `cd lab01`).

    Детальні інструкції щодо встановлення залежностей (якщо вони є) та запуску знаходяться у файлі `README.md` всередині кожної відповідної папки.
